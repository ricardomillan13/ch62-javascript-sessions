# TIFC-Javascript

Este repositorio contiene una colección de proyectos de JavaScript, diseñados como material de aprendizaje para un bootcamp. Cada proyecto aborda diferentes conceptos y funcionalidades de JavaScript, construyendo una base sólida para el desarrollo web.

## Proyectos

### js-01: Tipos de datos y variables

Este proyecto es una introducción a los fundamentos de JavaScript. Cubre los conceptos básicos necesarios para empezar a programar en este lenguaje.

**Temas cubiertos:**

*   Explicar los tipos de datos.
*   Explicar los principales tipos de datos en JavaScript: string, boolean, number, array, object, undefined y null.
*   Explicar las variables.
*   Declarar variables con nombres significativos.
*   Explicar la conversión implícita y explícita (type casting).
*   Convertir una variable en un tipo number.
*   Convertir una variable en un tipo string.
*   Convertir una variable en un tipo boolean.

### js-02: Declarar y usar funciones

En este proyecto, se profundiza en declarar y usar funciones.

**Temas cubiertos:**

*  Explicar las funciones y su importancia.
*  Definir e invocar funciones.
*  Declarar una función que no devuelve ningún valor.
*  Declarar una función que devuelve un valor.
*  Usar los parámetros de la función.
